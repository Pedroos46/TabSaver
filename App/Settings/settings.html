Privacidad


Activar para modo incognito:

Simple Tab Saver ahora mismo no esta habilitado para modo incognito.
Para usar Simple Tab Saver en modo incognito y, para evitar guarde sus sessiones de incognito, se necesita aprovar el permiso.


Un vez activado, uelva a la configuración para deshabitar que las sessiones en incognito se guarden en el historial.
Puede posponer este paso si no le interesa usar la extensión en modo incognito.

Recordatorio: Your private data will always be used locally.
